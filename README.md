# Hello-rails-back-end

- [Link-to-React-repo](https://github.com/sediqullahbadakhsh/hello-rails-front-end)

## Built With

- _**Ruby**_
- _**Rails**_
- _**PostgreSQL**_

## Getting Started

### Clone this repository

```bash
$ git clone https://github.com/sediqullahbadakhsh/hello-react-rails.git
$ cd hello-react-rails
```

### Install gems

```bash
$ bundle install
```

### Setup database

```bash
$ rails db:create
```

### Start server

```bash
$ rails s
```

Open http://localhost:3000/ in your browser

### Run tests

```bash
$ bundle exec rspec
```
